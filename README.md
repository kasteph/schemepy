schemepy
========

A toy Scheme interpreter written in Python.


what it does
--------
It evaluates:
* `if` statements
* `lambda` and `lambda` expressions
* some simple math
 
it can also `define` a variable and evaluate its expression if it has any.

what it doesn't do
--------
`set!`, `begin`, `proc`
